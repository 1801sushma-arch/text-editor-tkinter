# text-editor-tkinter

## 📌 Description
A simple GUI-based text editor built using Python Tkinter that allows users to create, open, edit, and save text files.

## 🚀 Features
- Create a new file
- Open existing text files
- Edit text content
- Save files
- Menu-based navigation

## 🧠 Concepts Used
- Tkinter (GUI development)
- File handling (read/write)
- Dialog boxes (file explorer)
- Event-driven programming

## ▶️ How to Run
1. Make sure Python is installed
2. Run:
   python text-editor.py
